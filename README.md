\# Sprint\_5 — Автотесты для Stellar Burgers



\## Описание проекта



Этот проект включает в себя автотесты для веб-приложения \*\*Stellar Burgers\*\* — онлайн-конструктора бургеров. Проект написан с использованием \*\*Python\*\* и \*\*Selenium\*\* для автоматизации тестирования функциональности веб-сайта.



\*\*Задачи, которые покрывают тесты\*\*:

1\. Регистрация нового пользователя.

2\. Вход в личный кабинет.

3\. Переход в личный кабинет и конструктор.

4\. Проверка функциональности конструктора бургеров.



Проект включает в себя следующие основные тесты:

\- Успешная регистрация.

\- Ошибки при неверном пароле.

\- Переходы между разделами сайта: «Булки», «Соусы», «Начинки».



\## Требования



Для работы с проектом вам необходимо:

\- Python 3.x

\- Selenium

\- Google Chrome или Mozilla Firefox
